# Admin setup

## Environment variables
- `ACCESS_CODES`: comma-separated codes
- `ADMIN_KEY`: admin password (stored in Railway)

## Admin panel
1. Go to bethemath.matesuite.ai
2. Click **admin** in the sidebar
3. Enter `ADMIN_KEY` and unlock

## Updating access codes
1. Generate codes (scripts/generate_access_codes.py)
2. In Railway, update `ACCESS_CODES` (comma-separated)
3. Save
