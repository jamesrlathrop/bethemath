About

Teacher / Parent Notes
Simple, ethical use: low-pressure practice + clear feedback.
  • Use 5–10 minutes as a warm-up (same band + track).
  • Ask: “What mistaken idea caused this?” then “What’s the clean fix?”
  • If a learner misses an item, read the hint together before the next one.
  • Rotate tracks or use All Tracks for mixed practice.
